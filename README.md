**React Image Gallery**
--

Image gallery example with React and Local Storage.

**Sample code for**  
- Image Gallery
- Image Light Box
- Pagination
- File Upload
- Local Storage

**Setup**
- Clone the repo  
`git clone https://github.com/ShahnawazAli93/react-image-gallery.git`
- Run installation  
`npm install`
- You are ready to roll.. :) Start the project by running   
`npm start`
